val load_id : Conex_io.t ->
       Conex_repository.t ->
       Conex_resource.identifier -> (Conex_repository.t, string) Pervasives.result
load_id io repo id reads id, either a team or an individual author, in repo. and verifies the read data.
val load_ids : ?ids:Conex_utils.S.t ->
       Conex_io.t ->
       Conex_repository.t -> (Conex_repository.t, string) Pervasives.result
load_ids ~ids io repo folds over ids (default: all ids present in io), applying load_id.
val load_janitors : ?valid:(Conex_resource.identifier -> Conex_resource.Digest.t -> bool) ->
       Conex_io.t ->
       Conex_repository.t -> (Conex_repository.t, string) Pervasives.result
load_janitors ~valid io repo reads the team janitors from io, and all team members. It verifies the data by doing two steps: first it uses only those public keys which fingerprints are provided via valid. If successful, it uses in a second step the resulting repository to verify the remaining janitors, followed by verification of the janitor team.
val verify_item : ?authorised:(Conex_utils.S.t -> bool) ->
       ?release:(Conex_resource.name -> bool) ->
       Conex_io.t ->
       Conex_repository.t ->
       Conex_resource.name -> (Conex_repository.t, string) Pervasives.result
verify_item ~authorised ~release io repo name reads and verifies the authorisation of the package name. All ids who are authorised are loaded into the repository (using load_ids). If authorised is true, the package data for name is read and verified, followed by all releases (filtered with release). Each release is read and verified.
val verify_diff : Conex_io.t ->
       Conex_repository.t ->
       string -> (Conex_repository.t, string) Pervasives.result
verify_diff io repository patch first parses the patch and applies it. The janitors of the old repository are loaded and their public key fingerprints are used to validate the janitors team of the repository with applied patch. All identifiers and packages of the repository with patch applied are verified, and additionally all modified resources have their monotonicity verified.
